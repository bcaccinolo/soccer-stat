# server.ex/

## Description

Small API server in Elixr to test the language.

## Launch the server

```
mix run --no-halt
```

## Launch the server while coding

```
iex -S mix
```

## Hot recompiling

```
iex> r Server
```

No need to restart anything.

```
mix deps.get
```
to fetch the packages
